<div class="lang-zh" markdown="1">

以下是我们开发并开源的部分研究软件。完整列表见我的 **[GitHub](https://github.com/fanchangjun)**。

- **FINDER**：基于深度强化学习的复杂网络关键节点识别框架，仅需在小规模网络上训练即可高效泛化到大规模网络。2020 年开源于 GitHub，目前 230+ stars。（[源码](https://github.com/fanchangjun/FINDER) · [论文](https://www.nature.com/articles/s42256-020-0177-2)，*Nature Machine Intelligence*, 2020）
- **DrBC**：基于图神经网络的介数中心性节点排序模型，端到端识别图中高介数中心性的关键节点。2019 年开源于 GitHub，目前 30+ stars。（[源码](https://github.com/fanchangjun/DrBC) · [论文](https://arxiv.org/abs/1905.10418)，*CIKM*, 2019）
- **DIRAC**：用于自旋玻璃基态计算的深度强化学习框架，可在小规模实例上训练并应用于任意大规模实例，并能增强各类热退火方法。2023 年开源于 GitHub，目前 20+ stars。（[源码](https://github.com/fanchangjun/DIRAC) · [论文](https://www.nature.com/articles/s41467-023-36363-w)，*Nature Communications*, 2023）

</div>

<div class="lang-en" markdown="1">

Below are some of the research software we have developed and open-sourced. See my **[GitHub](https://github.com/fanchangjun)** for the full list.

- **FINDER** (*FInding key players in complex Networks through DEep Reinforcement learning*): a deep reinforcement learning framework for finding key players in complex networks; trained on small graphs, it generalizes efficiently to large-scale networks. Released on GitHub since 2020, with 230+ stars. ([source code](https://github.com/fanchangjun/FINDER) · [paper](https://www.nature.com/articles/s42256-020-0177-2), *Nature Machine Intelligence*, 2020)
- **DrBC** (*Deep ranker for Betweenness Centrality*): a graph-neural-network ranking model that identifies high-betweenness-centrality nodes in large graphs in an end-to-end manner. Released on GitHub since 2019, with 30+ stars. ([source code](https://github.com/fanchangjun/DrBC) · [paper](https://arxiv.org/abs/1905.10418), *CIKM*, 2019)
- **DIRAC** (*Deep reinforcement learning for spIn-glass gRound-stAte Calculation*): a deep reinforcement learning framework for computing spin-glass ground states; trainable on small instances and applicable to arbitrarily large ones, and able to enhance any thermal annealing method. Released on GitHub since 2023, with 20+ stars. ([source code](https://github.com/fanchangjun/DIRAC) · [paper](https://www.nature.com/articles/s41467-023-36363-w), *Nature Communications*, 2023)
